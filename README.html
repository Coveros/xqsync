Required libraries:

  • Codehaus XStream
  • MarkLogic XDBC

Required properties:

  • one of: INPUT_PACKAGE, INPUT_PATH, INPUT_CONNECTION_STRING
  • one of: OUTPUT_PACKAGE, OUTPUT_PATH, OUTPUT_CONNECTION_STRING

Available properties:

Propertydefault valuenotes
DELETE_COLLECTION null In combination with XDBC_COLLECTION and OUTPUT_CONNECTION_STRING, delete the XDBC_COLLECTION on the OUTPUT_CONNECTION_STRING before beginning synchronization.
FATAL_ERRORStrueAll exceptions are fatal.
INPUT_CONNECTION_STRING null Input documents will come from this XDBC connection.
INPUT_PACKAGE null Input documents will come from this zip file path.
INPUT_PATH null Input documents will come from this filesystem path.
INPUT_START_POSITION null Use the numeric value of this property as the starting position for the sequence of input documents.
LOG_LEVELINFO java.util.logger.Level at which to log.
LOG_HANDLERCONSOLE,FILE java.util.logger log handlers with which to log.
OUTPUT_CONNECTION_STRING null Documents will be written to this XDBC connection.
OUTPUT_FORESTSnullPermitted output forest names.
OUTPUT_PATH null Output documents will be written to this filesystem path.
OUTPUT_PACKAGE null Output documents will be written to this zip file path.
READ_PERMISSION_ROLESnull Names of any roles to attach to output documents.
THREADS1Number of worker threads to spawn.
URI_PREFIXnull String to prepend to all output uris.
URI_SUFFIXnull String to append to all output uris.
XDBC_COLLECTION null In combination with INPUT_CONNECTION_STRING, all documents in the named collection will be transferred.
XDBC_QUERY null In combination with INPUT_CONNECTION_STRING, all uris returned by the query will be transferred. This sample query would transfer the first 100 documents: for $i in doc()[1 to 100] return base-uri($i)
XDBC_ROOT_DIRECTORY null In combination with INPUT_CONNECTION_STRING, all documents in the named directory will be transferred.
